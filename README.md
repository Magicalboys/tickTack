## 项目构建

```
pnpm install
```

## 文档贡献

本项目文档基于 `dumi` + `Ant Design主题包` 开发。

进入 `website/`，可做如下工作

* 修改 `.dumirc.ts` 配置文件。
* 编写 `Markdown` 文档。
* `dumi` 采用的 约定式路由，在 `docs`文件夹下新建 `.md` 文件即可创建路由。

启动

```
pnpm run docs
```
